# **App Name**: Webmarker

## Core Features:

- Marker Creation: Create a marker by CTRL+left clicking a text block or field on a webpage. This stores the webpage URL and the clicked text.
- Marker Storage: Store the webpage URL and the corresponding clicked text of each marker.
- Semantic Grouping: Utilize Gemini-3-Flash-Preview through the backend to categorize and group pins semantically and contextually. Gemini is used as a tool, so that similar pins can be listed together.
- Overlay Display: Display an overlay on the side of the browser (not the sidebar) to show the sorted and grouped markers.
- Navigation: Navigate back to the saved markers using the data which is being stored and shown in the overlay. Allow a visual guide.

## Style Guidelines:

- Primary color: Subtle blue (#75A9FF) to evoke calmness and focus.
- Background color: Light grey (#F0F4F8) for a clean and unobtrusive look.
- Accent color: Soft orange (#FFC275) for highlighting important markers and actions.
- Body font: 'PT Sans', a humanist sans-serif font. It is used for body text for comfortable reading and general usability.
- Headline font: 'Space Grotesk' a proportional sans-serif. This is for for titles.
- Simple and clear icons to represent different marker types and categories.
- Clean and organized layout for the overlay, with clear visual hierarchy for marker groups and individual markers.
- Subtle animations for marker creation and navigation to provide visual feedback.
